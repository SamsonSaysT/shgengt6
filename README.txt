OBS Countdown Timer for GitHub Pages

Countdown target:
November 18, 2026 at 11:59 PM Eastern Time / Detroit time.

Files included:
- index.html
- README.txt

Font setup:
Put your font file named exactly:
Pricedown Bl.otf

in the same folder as index.html before uploading to GitHub.
The font file itself is not bundled here, so use your original uploaded/local font file.

GitHub Pages setup:
1. Create a new GitHub repo.
2. Upload index.html and your font file, Pricedown Bl.otf, into the root of the repo.
3. Go to Settings > Pages.
4. Set Source to Deploy from a branch.
5. Choose main branch and root folder.
6. Your countdown will be available at your GitHub Pages URL.

OBS setup:
1. Add Browser Source.
2. Paste the GitHub Pages URL.
3. Set width/height to whatever fits your layout, like 1920x300.
4. Make sure custom CSS background is not forcing a background color.

Easy edits inside index.html:
- --text-color changes the text color.
- --font-size changes the size.
- --text-stroke can add an outline, example: 2px #000000
- --shadow can add a glow/shadow, example: 0 0 12px #000000
